# Lesson 2: Basic Operators and Data Types

In JavaScript, data comes in different forms (types), and we use operators to manipulate them.

## Common Data Types
- **String**: Text wrapped in quotes, e.g., `"Hello"`.
- **Number**: Integers or decimals, e.g., `42`, `3.14`.
- **Boolean**: Logical values, `true` or `false`.

## Common Operators
- **Arithmetic**: `+` (addition/concatenation), `-` (subtraction), `*` (multiplication), `/` (division).
- **Assignment**: `=` (assigns a value).

Example:
```javascript
let score = 10;
let bonus = 5;
let total = score + bonus; // Addition
let message = "Total score is: " + total; // Concatenation
console.log(message);
```
