# Lesson 5: Extra Loop Practice

It is completely normal to find loops a bit tricky at first! Let's break them down to build more confidence.

## Think Like a Computer
When you write a loop, think about three things:
1. **Starting Point**: Where does the counter start? (e.g., `let i = 1`)
2. **Ending Condition**: When should it stop? (e.g., `i <= 10`)
3. **Step**: How much does the counter change? (e.g., `i++` for +1)

## Practice Exercises
Try to create a file `js_loop_practice.js` and complete these:

1. **Countdown**: Use a `for` loop to print numbers from 10 down to 1.
2. **Summing**: Use a `while` loop to add up all numbers from 1 to 5. (e.g., 1 + 2 + 3 + 4 + 5 = 15).
3. **Multiplication**: Use a `for` loop to print the 5 times table up to 50 (5, 10, 15... 50).

Take your time with these! If you get stuck, explain to me which part feels confusing and we can talk through it together.
