## Lesson 1: Variables and Basic Output
JavaScript uses `let` and `const` for variable declarations.
- Use `let` for variables that might change.
- Use `const` for values that should remain constant.

Example:
```javascript
let greeting = "Hello, Masha!";
const course = "JavaScript";
console.log(greeting + " Welcome to " + course);
```
